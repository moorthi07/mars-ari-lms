Module 3 — Connecting to a Robot

Before you can control anything, you need to connect to a robot. Here's how.

Step 1: Open the Robots window

Click the menu icon → Management → Robots & Devices. You'll see a list of available robots.

Step 2: Check the robot's status

Each robot shows a status indicator:

  • 🟢 Online — Robot is connected and ready
  • 🟡 Busy — Another operator is using it
  • 🔴 Offline — Robot is not connected

Step 3: Check signal quality

Look at the signal meter in the video panel. You want at least 3 bars before starting a session. Poor signal = laggy controls = bad data.

Step 4: Check the camera feed

Before touching any controls, make sure you can see the robot's camera feed clearly. If the image is black, blurry, or frozen, don't start — report the issue.

What if the robot doesn't connect?

  • Refresh the page and try again
  • Check your internet connection
  • Try a different robot if available
  • Contact support if the issue persists
