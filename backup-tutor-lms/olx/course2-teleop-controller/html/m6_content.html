Module 6 — Voice Controls

Voice controls let you send commands to the robot by speaking. This is especially useful for language-conditioned AI training — the robot learns to associate your spoken instructions with the actions you perform.

Setting Up Voice Control

  1. Click the microphone icon in the controls panel
  2. Allow browser microphone access when prompted
  3. Speak clearly into your microphone
  4. Your speech is transcribed automatically (Whisper AI)

Live Voice Control

Live voice control sends commands in real-time as you speak. The robot responds to recognized commands immediately. Useful for hands-free operation when both hands are busy with other controls.

When to Use Voice vs Manual

  • Use voice when you want to label what you're doing ("picking up red cube", "moving to bin")
  • Use manual for precise movements that require fine motor control
  • Use both for the best training data — speak what you're doing while doing it

Good Voice Data Habits

  • Speak clearly and at normal pace
  • Describe the task, not the controls ("pick up the cube" not "move arm left")
  • Be consistent — use the same phrases for the same actions
  • Minimize background noise
