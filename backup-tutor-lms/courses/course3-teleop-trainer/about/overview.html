TeleOp Trainer Certification

Collect better data. Train better robots.

This course takes you from robot operator to AI trainer. You'll learn how to design data collection sessions, curate and label datasets, configure VLA/VLM models, and run the full Physical AI iteration loop. Prerequisite: TeleOp Controller Certification.

What you'll be able to do

  • Design structured data collection sessions for AI training
  • Understand and work with telemetry, video, and audio data
  • Curate and label robot datasets
  • Use simulation to generate training data at scale
  • Configure VLA and VLM models for your robot tasks
  • Run the full collect → train → evaluate → improve loop

Prerequisites

TeleOp Controller Certification (CTRL101)

Duration

11 modules · ~5 hours · Self-paced · Certificate on completion
